Proxy URL siap digunakan di StreamElements
1 Proxy URL untuk Widget SE

Ini adalah URL yang kamu masukkan ke field proxyUrl di StreamElements.

—
⚠ URL ini mengandung token Spotify kamu. Jangan bagikan ke orang lain. Token akan diperbarui otomatis setiap jam.
2 Cara pasang di StreamElements

Di widget editor SE, buka tab Fields dan tambahkan:

Field name: proxyUrl Type : text Default : —

Lalu di bagian JS widget, pastikan ada baris ini di awal:

var _p = new URLSearchParams(location.search); if (_p.get('proxyUrl')) CFG.proxyUrl = _p.get('proxyUrl');
